# EntreSkill Hub

A full‑stack web application that helps users discover business ideas based on their skills, provides step‑by‑step roadmaps, learning resources, and mentor connections.

## Features (initial scaffold)
- React front‑end (Vite) with responsive UI
- Node.js + Express back‑end
- MongoDB data store
- JWT‑based authentication (user & mentor login)
- Basic CRUD APIs for users, mentors, business ideas, and roadmaps

## Résumé Highlights
- Designed and implemented a complete MERN‑style application from scratch
- Integrated secure JWT authentication and role‑based access control
- Built modular front‑end architecture with reusable React components
- Configured MongoDB schema design for flexible business‑idea data model
- Prepared CI‑ready project structure ready for deployment on Vercel/AWS
